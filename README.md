# Dashboard de Vendas no Excel

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio de criação de um dashboard de vendas no Excel.

O objetivo principal foi transformar dados brutos em informações visuais claras e organizadas, permitindo uma análise eficiente do desempenho de vendas e auxiliando na tomada de decisões.

O dashboard foi construído utilizando recursos do Microsoft Excel, incluindo:

* Tabelas dinâmicas
* Segmentação de dados
* Gráficos dinâmicos
* Fórmulas e cálculos
* Indicadores visuais (KPIs)
* Organização visual para melhor análise dos dados

---

## 📊 Informações Apresentadas no Dashboard

O dashboard contém informações como:

* Total de vendas
* Receita total
* Produtos mais vendidos
* Evolução das vendas
* Comparativo entre categorias
* Indicadores de desempenho
* Filtros interativos
