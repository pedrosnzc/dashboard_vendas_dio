## 📊 Sobre o Projeto

Projeto desenvolvido para transformar dados brutos em informações visuais utilizando Excel.

O dashboard apresenta:
- Receita total
- Total de vendas
- Indicadores de desempenho
- Gráficos dinâmicos
- Filtros interativos

## 🛠 Ferramentas Utilizadas

- Microsoft Excel
- GitHub
